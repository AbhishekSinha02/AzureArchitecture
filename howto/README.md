## How to Use

Each file answers the question an interviewer *actually* asks when probing hands-on depth.  
Every answer is 3–6 sentences — enough to sound like you've done it, short enough to say confidently.  
Every answer ends with a **Deep Dive Hint** — a thread you can open in a follow-up session.
